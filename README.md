# Xtream-UI Ubuntu 20.04

## Requirements
- **Ubuntu 20.04**


## Installation

```bash
sudo apt update && rm -rf install.py && wget -qO install.py https://raw.githubusercontent.com/silenttrixx/Xtream-Ui_silenttrix_ck_20.04/main/install.py && sudo python3 install.py
```
